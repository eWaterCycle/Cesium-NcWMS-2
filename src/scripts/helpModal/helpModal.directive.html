eWaterCycle visualization with Cesium-NcWMS

WARNING: You are using a mobile client, so we offer limited functionality. Please use a desktop webbrowser for the best user experience we can offer.

Welcome!

This is the Beta preview of the eWaterCycle Global Hydrological forecast system. See our website for more info.

Please note that since we're still in beta, the data shown here is only an indication. Please do not make real-life decisions based on the information here.

Also note that the beta version of eWaterCycle uses a supercomputer that is shared with other academic research projects. Due to queing reasons, a daily forecast may be delayed, even up to a few days, when other projects temporarily need the supercomputer capacity.

Features

The top-right menu

The top-right menu is where you can configure your map view. It offers several dropdown menus, from left to right:

  • The "Fly To" menu: Use this menu to 'fly' to the country you want to focus on.
  • The "View" menu: Use this menu to toggle between globe, columbus, and map views.
  • The "Dataset" menu: Select the dataset you want to see on the globe.
  • The "Palette" menu: Select the palette you want to as a colormap for your dataset.
  • The "Log" button: Toggle between logarithmic and linear scaling of the colormap.
  • The "Terrain" button: Toggle between terrain settings, transparent colormap with terrain, or terrain-less and opaque.
  • The "Borders" button: Toggle country borders on/off.
The legend

The legend on the right side offers two textboxes, where you are able to set the minimum and maximum value between which you want the colormap to scale.

The time bar

The time bar on the bottom is used to define the time you want to see data of. it is automatically scaled to the extents of your selected dataset.

Data graphs

The graphs on the left side appear when the user clicks the button to display them. Clicking a location on the map with sufficient time-based data will then display a graph of the data in that location.

The graphs are plotted as a plume where the colors of the plume represent one and two standard deviations respectively around the mean value. When the standard deviation is large compared to the mean flow this can lead to errors where negative flow is displayed in the visualization.

Troubleshooting

If you can only see a globe with normal coloring, and the dropdown menus at the top seem suspiciously empty, that probably means you have no ncWMS server attached, or that server is serving no data at the moment. Please refer to the README file.
